# PaleNote
Plataforma para processamento de notas fiscais eletrônicas (NF-e), validação de CPF/CNPJ, geração de PDF e processamento assíncrono.

## Por que PaleNote?

O nome faz referência ao desgaste natural de documentos físicos ao longo do tempo. Notas impressas tendem a perder contraste, amarelar e ter informações apagadas. O PaleNote foi concebido para representar a transição desses documentos para um ambiente digital confiável, onde os dados permanecem íntegros, rastreáveis e acessíveis.
